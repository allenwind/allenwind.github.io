当我们研究和理解一家企业时，往往要从企业的基本问题出发，如业务和产品、资产模式和商业模式等等。但是往往面临浩如烟海的资料、缺乏结构化的信息、缺乏轻重重要性的信息。

那么，有什么方法可以让我们快速理解一家企业，对其建立基本理解？

这里有一个比较便捷的做法，利用基本面投资框架构建 prompt 模版，借助 LLM 辅助我们快速理解一家企业。

投资框架与 prompt

下面是我提炼的投资框架 prompt：

假设你是商业专家和基本面分析专家。理解一家企业可以从以下的框架和问题出发：

  • 基本介绍
  • 主要业务和主要产品有哪些？
  • 微笑曲线：企业处于微笑曲线的位置？
    • 左段为技术、专利
    • 中段为组装、制造
    • 右段为品牌、服务
  • 资产模式：企业的资产模式（轻资产、重资产与无形资产）是情况是什么？
  • 商业模式：从标准化和规模化这两个维度刻画这家企业？
    • 产品型：高度标准化与规模效应
    • 平台型：存在规模效应与正向反馈
    • 订单型：定制化程度高与一定规模效应
    • 项目型：非标准化与不存在规模效应
  • 竞争优势或防御机制：企业属于品牌知名度、规模效应、技术壁垒、网络效应中哪种类型？
  • 市场特点：企业所处行业的市场表现的驱动核心主导因素是价差因素、销量因素、价格因素、政策因素中哪种？
    • 价格主导：供给弹性低或需求弹性低
    • 价差主导：原料加工行业，成本波动较大
    • 销量主导：需求弹性较高或者价格变化不大
    • 政策主导：行业政策周期明显
  • 企业增长特点和方向：
    • 是否具备持续的提价能力？
    • 企业的二次增长方向在新技术、新效益、新需求、新业务中注重哪种？
  • ESG 特征画像：从环境、社会、治理这三个维度描述这家企业做了哪些事情？

这个框架有企业特征 + 问题构成。请基于这个框架描述「具体企业名字」这家企业。你的输出要按照框架的特征维度组织。

通过以上 prompt，把「具体企业名字」改为需要研究的企业即可。

例子一：比亚迪

以下是对比亚迪公司进行基于框架和问题的分析：

  1. 基本介绍比亚迪是一家中国领先的新能源车辆制造商，成立于1995年，总部位于深圳。公司以电池制造起家，逐渐拓展到汽车、新能源等领域。比亚迪是全球最大的电动汽车制造商之一，并且在混合动力汽车市场也有深厚的技术基础。
  2. 主要业务和主要产品有哪些？比亚迪的主要业务包括汽车制造、电池制造、太阳能板制造等。主要产品包括电动汽车、混合动力汽车、电池组、太阳能板等。
  3. 微笑曲线：企业处于微笑曲线的位置？比亚迪处于微笑曲线的右段，主要生产和销售新能源汽车和电池，同时也在研发和设计方面具有很强的实力。公司注重产品的品牌和服务，通过不断创新和提高品质来提升竞争力。
  4. 资产模式：企业的资产模式（轻资产、重资产与无形资产）是情况是什么？比亚迪属于重资产企业，主要资产包括土地、厂房、设备等固定资产。同时，公司也拥有大量的无形资产，如知识产权、品牌价值等。
  5. 商业模式：从标准化和规模化这两个维度刻画这家企业？比亚迪的产品包括新能源汽车、电池等，这些产品都有一定的标准化程度，可以进行规模化生产。同时，公司也注重产品的研发和创新，以满足不同客户的需求。
  6. 竞争优势或防御机制：企业属于品牌知名度、规模效应、技术壁垒、网络效应中哪种类型？比亚迪作为全球最大的电动汽车制造商之一，具有较高的品牌知名度和规模效应。同时，公司在电池技术方面具有领先地位，具有一定的技术壁垒。此外，公司也注重构建网络效应，通过不断扩大用户基础和提升用户体验来增强竞争力。
  7. 市场特点：企业所处行业的市场表现的驱动核心因素是价格因素、价差因素、销量因素、政策因素中哪种？新能源汽车市场的驱动核心因素是政策因素。政府对于新能源汽车的支持和鼓励政策是推动市场发展的主要动力之一。此外，随着环保意识的提高和油价的不断上涨，消费者对于新能源汽车的需求也在不断增加。
  8. 企业增长特点和方向：比亚迪一直在积极拓展新的业务领域和市场，包括海外市场和新的能源领域。公司注重研发和创新，不断推出新的技术和产品来满足市场的需求。同时，比亚迪也致力于提高生产效率和降低成本，以保持竞争优势。
  9. ESG 特征画像：从社会、环境、治理这三个维度描述这家企业做了哪些事情？比亚迪注重社会责任和可持续发展。公司在环保方面积极采取措施，推动新能源汽车的发展，减少碳排放。同时，公司也关注员工福利和社会责任，积极参与公益事业和社会活动。在治理方面，比亚迪注重规范运作和透明度，建立了健全的治理结构和规章制度。

例子二：漫步者

漫步者 是一家以生产高品质音响设备为主的企业，总部位于中国。下面按照您提供的框架对漫步者进行描述：

基本介绍：

漫步者是一家在音响设备行业具有广泛知名度的企业，成立于1996年，总部位于北京。该公司主要生产和销售高品质的音响设备，包括扬声器、功放、解码器等产品。

主要业务和主要产品：

漫步者的主要业务是生产和销售高品质的音响设备。其主要产品包括扬声器、功放、解码器等，这些产品广泛应用于家庭、商务、汽车等领域。

微笑曲线：

漫步者处于微笑曲线的左段，即技术、专利的位置。该公司在音响设备领域拥有多项专利技术，并且注重产品的研发和创新。

资产模式：

漫步者属于重资产型企业，其主要资产包括土地、厂房、机器设备等。此外，该公司的无形资产也较为丰富，包括专利技术、品牌形象等。

商业模式：

漫步者属于产品型模式的企业。该公司的产品具有高度的标准化和规模效应，通过大规模生产来降低成本和提高效率。

竞争优势或防御机制：

漫步者的竞争优势主要包括品牌知名度、规模效应和技术壁垒。该公司在音响设备领域具有较高的市场份额，并且注重产品的品质和售后服务。此外，漫步者还拥有多项专利技术，使得其他企业难以进入该市场。

市场特点：

漫步者所处的音响设备市场的驱动核心因素是价格因素和价差因素。由于消费者对音响设备的需求弹性较低，因此价格和价差是影响市场表现的主要因素。

企业增长特点和方向：

漫步者具有较强的提价能力，并且已经开始向新技术和新业务方向发展。该公司在研发和创新方面投入了大量的资金和人力，致力于推出更加高品质和差异化的产品和服务。

ESG 特征画像：

漫步者在环境方面注重节能减排和绿色生产，同时也在社会方面积极履行企业社会责任。该公司在治理方面也具有较为完善的公司治理结构和风险控制机制。

例子三：腾讯

基于给定的框架和问题，以下是腾讯这家企业的描述：

一、基本介绍
腾讯是中国最大的互联网综合服务提供商之一，成立于1998年，总部位于深圳。公司一直致力于提供优质、高效、便捷的互联网服务，包括社交、游戏、金融、广告等多个领域。

二、主要业务和主要产品
腾讯的主要业务包括社交平台、游戏、金融、广告等。主要产品包括微信、QQ、腾讯云、腾讯游戏等。其中，微信是腾讯最著名的产品之一，是中国最受欢迎的社交平台之一。

三、微笑曲线
腾讯在微笑曲线的左段和右段都有布局。在左段，公司拥有一些知名技术专利和强大的研发能力。在右段，腾讯拥有庞大的用户群体和强大的品牌影响力，通过提供各种互联网服务获取收益。

四、资产模式
腾讯的资产模式属于轻资产和无形资产为主。公司的固定资产规模不大，但无形资产占比很高，包括品牌、专利、人才等。

五、商业模式
腾讯的商业模式属于平台型。公司通过提供免费的社交平台和优质的游戏体验吸引用户，然后通过广告和游戏内购买等方式获取收益。这种模式具有规模效应和正向反馈的特点。

六、竞争优势或防御机制
腾讯的竞争优势主要包括品牌知名度、规模效应和技术壁垒。公司在中国互联网市场拥有很高的市场份额，具有很强的品牌影响力和用户黏性。同时，腾讯在技术方面也具有很强的实力，能够自主研发出很多先进的技术和产品。

七、市场特点
腾讯所处行业的市场特点属于销量主导。公司的主要产品是社交平台和游戏，这些产品的需求弹性较高，用户数量的多少直接决定了公司的收益。因此，腾讯需要不断推出新的产品和服务来吸引用户，提高用户黏性。

八、企业增长特点和方向
腾讯的增长特点主要是通过不断推出新的产品和服务来实现。公司注重新技术和新业务的研发和应用，不断推出新的游戏、社交平台和其他服务。同时，腾讯也注重提高产品质量和服务水平，不断满足用户需求和提高用户体验。未来，腾讯将继续致力于提供更好的互联网服务，扩大市场份额和提高用户满意度。

九、ESG 特征画像
从环境、社会、治理这三个维度来看，腾讯注重环保和可持续发展。公司积极推广节能减排措施，采用绿色办公和生产方式，减少对环境的负面影响。在社会方面，腾讯注重社会责任和公益事业，通过各种方式回馈社会。在治理方面，腾讯注重规范治理和合规经营，建立完善的管理制度和监管机制，保障公司和用户的利益。

总结

通过以上 prompt 结合 LLM 可以快速理解企业的基本信息。

此外，还可以通过交互或拆分子维度进一步提问。当然，如果你使用的 LLM 支持插件，通过商业信息插件获得的信息更可靠。
